# 0.22.1 — Ajuste de espaçamento do card de ritual

## Objetivo

Reduzir o espaçamento vertical entre os cards internos do ritual depois do refactor estrutural da `0.22.0`.

## Mudanças

- Diminui o gap entre:
  - **Conjuração**
  - **Dano**
  - **Efeito**
- Mantém a moldura externa removida.
- Não altera espaçamento interno da resistência ou das ações.
- Não altera comportamento de dano, resistência, efeito ou estados dos botões.

## Fora de escopo

- Multi-target.
- Template Regions.
- Mudanças de regra.
- Novo refactor estrutural.
