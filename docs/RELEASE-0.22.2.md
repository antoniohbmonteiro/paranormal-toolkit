# 0.22.2 — Alinhamento visual do card de Efeito

## Objetivo

Deixar o card **Efeito** com a mesma régua visual dos cards **Conjuração** e **Dano**.

## Mudanças

- Troca o espaçamento entre cards internos para um `gap` único no roll card estruturado.
- Reduz o gap vertical entre **Conjuração**, **Dano** e **Efeito** para `0.18rem`.
- Remove margens verticais próprias dos filhos diretos do roll card estruturado.
- Ajusta a tipografia do título **Efeito** para parecer irmão dos títulos **Conjuração** e **Dano**.
- Ajusta levemente a linha do label do efeito para manter o mesmo ritmo visual.
- Mantém a paleta dourada do card de efeito.

## Fora de escopo

- Mudanças de regra.
- Multi-target.
- Template Regions.
- Alteração em dano, resistência, Active Effects ou estados dos botões.
