# Paranormal Toolkit 0.23.2

## Correções visuais no card multi-target

- O card inteiro de cada alvo agora alterna aberto/fechado quando clicado, exceto nos botões internos.
- A setinha virou indicador visual de expansão, sem aparência de botão.
- O dano do alvo fechado ficou compacto para caber melhor no chat:
  - `⚡ 13 PV` para dano normal.
  - `🛡 ½ 6 PV` para dano reduzido futuramente.
- O detalhe aberto mantém espaço para labels completos e explicação da resistência.

## Observação

Esta versão ainda mantém o multi-target como base visual/modelo. A rolagem e aplicação por alvo ficam para a próxima etapa funcional.
